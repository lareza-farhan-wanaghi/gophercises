## Description

This repository contains my solutions for the [gophercises](https://gophercises.com/) exercises.


### Folder details

**1. quiz**

Solution for the 1st exercise. Activities involved: making several functions to do some functionalities, including reading CSV files and flags, listening to user inputs, and simulating a timer. [Learn more](https://github.com/gophercises/quiz)


**2. urlshort**

Solution for the 2nd exercise. Activities involved: making a simple HTTP listener, its handler functions, and functions to read and parse JSON & YAML files and establish a connection to a database. [Learn more](https://github.com/gophercises/urlshort)


**3. cyoa**

Solution for the 3rd exercise. Activities involved: creating go template files and functions to parse them and an HTTP handler function that renders the parsed go templates. [Learn more](https://github.com/gophercises/cyoa)

**4. link**

Solution for the 4th exercise. Activities involved: creating functions to parse an HTML file and collect the href values and inner texts of all "a" tags within the specified HTML file and writing test functions to test the functions created. [Learn more](https://github.com/gophercises/link)
